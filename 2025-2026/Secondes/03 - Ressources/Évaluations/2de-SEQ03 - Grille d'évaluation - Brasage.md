## CC2 — Organiser la réalisation ou l'intervention

> *Évaluée principalement en SÉA01 (organisation du poste, EPI) et en entrée de SÉA03.*

| Critère (en gras) / Indicateur observable                                                                      | A   | NE  | 🔴  | 🟠  | 🟡  | 🟢  |
| -------------------------------------------------------------------------------------------------------------- | --- | --- | --- | --- | --- | --- |
| **Organiser son poste de travail en assurant la sécurité de tous les intervenants**                            |     |     |     |     |     |     |
| Le poste est préparé avant démarrage : gabarit, schéma et outillage disposés avant de brancher le fer          |     |     |     |     |     |     |
| Le fer à braser est posé sur son support dès qu'il n'est pas utilisé                                           |     |     |     |     |     |     |
| **Identifier les EPI adaptés à l'intervention et les porter**                                                  |     |     |     |     |     |     |
| L'élève identifie les EPI obligatoires (lunettes, blouse, ventilation) et les relie aux risques correspondants |     |     |     |     |     |     |
| Les EPI sont portés pendant toute la durée de la manipulation, sans rappel de l'enseignant                     |     |     |     |     |     |     |

## CC4 — Réaliser une installation ou une intervention

> *Évaluée en SÉA03 : brasage des trois chaînes du feu tricolore voiture.*

| Critère (en gras) / Indicateur observable                                                                                  | A   | NE  | 🔴  | 🟠  | 🟡  | 🟢  |
| -------------------------------------------------------------------------------------------------------------------------- | --- | --- | --- | --- | --- | --- |
| **Implanter et câbler les composants conformément au dossier de fabrication**                                              |     |     |     |     |     |     |
| Les composants sont implantés aux positions du gabarit, dans l'ordre recommandé (résistance → LED → fil)                   |     |     |     |     |     |     |
| La polarité de chaque LED est vérifiée avant brasure (anode du côté indiqué sur le gabarit)                                |     |     |     |     |     |     |
| **Réaliser un produit conforme aux spécifications — gestes techniques corrects**                                           |     |     |     |     |     |     |
| Les brasures sont brillantes, coniques et lisses — pas de brasure froide ni de pont entre deux pistes                      |     |     |     |     |     |     |
| La continuité de chaque chaîne est vérifiée au multimètre avant le test sous tension                                       |     |     |     |     |     |     |
| **Réaliser un produit conforme aux spécifications — résultat fonctionnel**                                                 |     |     |     |     |     |     |
| Les trois LEDs voiture s'allument lors du test (rouge, orange, verte)                                                      |     |     |     |     |     |     |
| Une seule LED est allumée à la fois (pas de court-circuit entre pistes)                                                    |     |     |     |     |     |     |
| **Opérer avec une attitude écoresponsable**                                                                                |     |     |     |     |     |     |
| Les chutes de pattes de composants sont collectées et jetées dans la poubelle dédiée (pas laissées sur le plan de travail) |     |     |     |     |     |     |

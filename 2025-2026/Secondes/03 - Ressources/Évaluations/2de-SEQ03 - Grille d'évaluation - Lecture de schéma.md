## CC4 — Réaliser une installation ou une intervention

> *La lecture de schéma est évaluée ici comme compétence préparatoire à la réalisation : un élève qui ne sait pas lire le schéma ne peut pas braser correctement en SÉA03.*

| Critère (en gras) / Indicateur observable                                                                                             | A   | NE  | 🔴  | 🟠  | 🟡  | 🟢  |
| ------------------------------------------------------------------------------------------------------------------------------------- | --- | --- | --- | --- | --- | --- |
| **Implanter les composants conformément au dossier — identifier les composants et leur rôle**                                         |     |     |     |     |     |     |
| Tous les composants du schéma sont identifiés et associés à leur symbole normalisé (LED, résistance, bouton, Arduino)                 |     |     |     |     |     |     |
| La valeur de chaque composant est correctement relevée (220 Ω, 10 kΩ, couleur LED)                                                    |     |     |     |     |     |     |
| **Implanter les composants conformément au dossier — lire les connexions**                                                            |     |     |     |     |     |     |
| Le tableau de correspondance est entièrement complété et correct (référence schéma → composant réel → broche Arduino)                 |     |     |     |     |     |     |
| L'élève identifie correctement la cathode et l'anode sur le symbole schéma, et les relie aux repères physiques (patte courte, méplat) |     |     |     |     |     |     |
| L'élève distingue broche logique Arduino (D2, D3…) et position physique sur le connecteur                                             |     |     |     |     |     |     |
| **Réaliser un produit conforme — planifier l'implantation sur perfboard**                                                             |     |     |     |     |     |     |
| Le gabarit perfboard indique la position de tous les composants                                                                       |     |     |     |     |     |     |
| La polarité de chaque LED (A / K) est correctement reportée sur le gabarit                                                            |     |     |     |     |     |     |
| L'espacement entre composants est suffisant pour permettre le brasage                                                                 |     |     |     |     |     |     |
